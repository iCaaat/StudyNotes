# 常用语言命名规范

## Java

| 位置   | 命名规范                                       | 说明 |
| ------ | ---------------------------------------------- | ---- |
| 类名   | 大驼峰命名，如`UserDetails`                    |      |
| 接口名 | 大驼峰命名，如`IUserDetails`                   |      |
| 方法名 | 第一个单词小写，后续单词大写，如`getUser`      |      |
| 变量名 | 第一个单词小写，后续单词大写，如`userName`     |      |
| 常量名 | 全大写+下划线，如`MAX_USERS`                   |      |
| 包名   | 全小写，域名倒序+项目名，如`com.example.myapp` |      |



## SQL数据库

| 位置   | 命名规范                           | 说明 |
| ------ | ---------------------------------- | ---- |
| 数据表 | 全小写+数字+下划线，单数，如`user` |      |
| 列名   | 全小写+数字+下划线                 |      |



## mybatis

| 位置                 | 命名规范                                     | 说明 |
| -------------------- | -------------------------------------------- | ---- |
| Mapper接口的形参规范 | 遵循java规范，多参数使用`@Param("名字")`注解 |      |
| XML文件中列名        | 与数据库命名保持一致                         |      |



## C语言

| 类型     | 命名风格                | 示例                   |
| -------- | ----------------------- | ---------------------- |
| 文件名   | 小写+下划线             | `link_list.c`          |
| 变量名   | 小写+下划线             | `elem_count`           |
| 函数名   | 小写+下划线             | `init_list`            |
| 结构体名 | 小写+下划线或PascalCase | `struct node` / `Node` |
| 类型定义 | 首字母大写或全大写      | `ElemType` / `Node`    |
| 常量名   | 全大写+下划线           | `MAX_SIZE`             |
| 宏定义   | 全大写+下划线           | `SUCCESS`              |
| 枚举名   | 首字母大写+全大写值     | `STATUS_OK`            |
| 全局变量 | 小写+下划线+前缀        | `g_list_head`          |
| 局部变量 | 小写+下划线             | `temp_node`            |
| 指针变量 | 小写+下划线+后缀        | `node_ptr`             |
