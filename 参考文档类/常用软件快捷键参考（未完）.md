# 常用软件快捷键参考

## IDEA快捷键参考

|                      快捷键                       |        使用处        |                 作用                 | 备注 |
| :-----------------------------------------------: | :------------------: | :----------------------------------: | :--: |
| **Ctrl+shift+Minus(-)** 和 **Ctrl+shift+Plus(+)** |                      |         快速折叠/展开代码块          |      |
|                **Alt**+**Insert**                 |                      | 快速生成代码（如getter和setter方法） |      |
|                    **Ctrl+D**                     |                      |         快速复制此行到下一行         |      |
|                    **Ctrl+M**                     |                      |      选取区域快速生成为一个方法      |      |
|                **Ctrl + Alt + L**                 |                      |            快速格式化代码            |      |
|                **Ctrl + Alt + M**                 |                      |             快速抽取方法             |      |
|                   **Ctrl + P**                    | 在方法调用处的括号处 |             显示方法参数             |      |
|                **Ctrl + Alt + <**                 |                      |              返回上一步              |      |
|                **Ctrl + Alt + T**                 |  选中需要包裹的代码  |            使用xx方法包裹            |      |
|                  **Ctrl + f12**                   |                      |             类内查找方法             |      |
