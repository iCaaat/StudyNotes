# 10.表格

使用竖线分隔表单项，第一行表单内为表头名，只在第二行表单内使用分隔线---隔开，之后如表头形式书写。

若想要控制对其方式，左对其则在左边加英文冒号“：”，右对齐则在右边加冒号，居中则两边都加冒号（靠近竖线处“|”）。
